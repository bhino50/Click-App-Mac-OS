Click Download

macOS menu bar utility

Mechanical keyboard sounds as you type.

Click lives in your menu bar and plays low-latency key sounds with per-key variation, velocity sensitivity, and Mechvibes-compatible packs. No subscriptions, no telemetry.

Download for macOS Version 1.0 · macOS 14+

Install

Download, drag to Applications, grant Accessibility.

Open the DMG, read Install First — Read Me.txt, drag Click to Applications, then launch. Click is open source and not Apple-notarized yet, so macOS blocks the very first launch — expected, and you only bypass it once:

  • macOS 13–14: right-click Click in Applications → Open → confirm Open.
  • macOS 15+: double-click once (blocked), then System Settings → Privacy & Security → Open Anyway.

A welcome window opens automatically and walks you through Accessibility access. Use Test sound in the menu bar panel to verify audio before granting permission. Reopen the guide anytime via Setup Guide… in the menu.
