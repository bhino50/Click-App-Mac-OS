Click — Install & First Launch
==============================

1. Drag Click.app into the Applications folder.

2. FIRST OPEN (macOS Gatekeeper)
   Click is open source and not Apple-notarized yet, so macOS blocks the
   first launch. This is expected — not a virus warning.

   macOS 13–14 (Ventura / Sonoma):
     • Right-click (or Control-click) Click in Applications
     • Choose Open, then click Open in the dialog

   macOS 15+ (Sequoia / Tahoe):
     • Double-click Click once (it will be blocked)
     • Open System Settings → Privacy & Security
     • Scroll to “Click was blocked…” and click Open Anyway

3. IN-APP SETUP
   A welcome window opens automatically and walks you through Accessibility
   access. Reopen it anytime from the menu bar keyboard icon → Setup Guide…

4. ACCESSIBILITY PERMISSION
   Click needs Accessibility to hear keystrokes globally. Keystrokes are
   never stored or transmitted. Enable Click in:
   System Settings → Privacy & Security → Accessibility

5. MENU BAR
   Click lives in the menu bar only (no Dock icon). Look for the keyboard
   icon near the clock. Use “Test sound” to verify audio before granting
   Accessibility.
